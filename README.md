# Weather App

Проста уеб апликация за показване на времето (HTML/CSS/JS).

## Описание

Този малък проект съдържа фронтенд пример, който показва текущото време за даден град.

## Файлова структура

- `index.html` — основна HTML страница
- `styles.css` — стилове
- `script.js` — логика на приложението

## Стартиране

Можете да отворите директно `index.html` в браузъра. За локален сървър (по-добре за някои браузъри и API заявки) използвайте:

```bash
python3 -m http.server 8000
# или
npx http-server . -p 8000
```

После отидете на `http://localhost:8000`.

## Конфигурация

Ако приложението използва външни APIs (например OpenWeather), добавете вашия API ключ в `script.js` или в подходящ конфигурационен файл.
